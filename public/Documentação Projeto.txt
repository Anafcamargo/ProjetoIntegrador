Documentação do Sistema CHAMAKI

Visão Geral
Um Website desenvolvido para conectar pessoas que precisam de ajuda tecnológica, com pessoas capacitadas e disponiveis para ajudar.

Funcionalidades:
Tela inicial
- Apresentação inicial
- Ranking com as melhores avaliações
- Informações de como funciona
- Informações sobre projeto 
- Cadastro 
- Login
- Cadastro de chamado

Como usar:
Cliente:
- Ao acessar a página temos as principais informações na tela inicial, depois para realizar um chamado será necessário fazer um cadastro no site para cliente.
- Efetuar Login
- Cadastrar chamado na página, descrevendo o problema e aguardar algum voluntário ajudar.

Voluntário:
- Cadastro para voluntários que queiram ajudar
- Efetuar Login
- Visualizar chamados em aberto e atender quais forem possíveis.
